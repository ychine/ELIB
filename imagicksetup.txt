### Step 1: Uninstall Current ImageMagick
1. Go to **Windows Control Panel > Programs and Features** (or search "Add or remove programs").
2. Find and uninstall "ImageMagick" (it might show as "ImageMagick 7.1.2-Q16-HDRI" or similar).
3. Delete any leftover folders, e.g., `C:\Program Files\ImageMagick-7.1.2-Q16-HDRI` (adjust based on your install path).
4. Edit your system Environment Variables:
   - Search for "Environment Variables" in Windows Start.
   - Under **System variables**, find **Path**, click Edit.
   - Remove any entries pointing to ImageMagick (e.g., `C:\Program Files\ImageMagick-7.1.2-Q16-HDRI`).
5. Restart your computer to clear any cached paths.

### Step 2: Download and Install Matching ImageMagick 7.1.0
1. Download the Windows 64-bit DLL installer for ImageMagick 7.1.0-18 Q16-HDRI from this direct link (hosted on GitHub via Npackd, a trusted open-source package repo):
   - **Download URL**: https://github.com/tim-lebedkov/packages/releases/download/2021_12/imagemagick64-7.1.0.18-ImageMagick-7.1.0-18-Q16-HDRI-x64-dll.exe

   - (If the link fails, alternative mirror: Search "ImageMagick 7.1.0-18 Q16 HDRI x64 dll exe" on Google and grab from npackd.org or similar archives.)
